# Getting Started with MDX in Next.js

MDX lets you use JSX in your markdown content. You can **import components**, such as interactive charts
or alerts, and embed them within your content.

## Why MDX?

- Write markdown with JSX
- Import and use React components
- Create interactive blog posts
- Style with Tailwind CSS

<Callout type="info">
  MDX is perfect for technical blog posts because you can include interactive examples!
</Callout>

## Code Example

```typescript
function HelloWorld() {
  return (
    <div className="p-4 rounded-lg bg-gray-100">
      <h1>Hello, MDX!</h1>
    </div>
  );
}
```
